# [풀이 코드](./BOJ_S4_11047.java)

<br>

간단한 그리디 문제이다. 가치가 가장 큰 동전부터 사용하여 가치의 합을 K로 만들기 위해 필요한 동전의 개수를 구한다.

<br>

## 그리디

```java
        int answer = 0;
        for (int i = N - 1; i > -1; --i) {
            int count = K / coins[i];
            answer += count;
            K -= coins[i] * count;
        }
```

배열 끝에서부터 순회하면서 가치가 높은 동전부터 사용할 수 있는 동전의 개수를 계산한다. 해당 동전을 사용할 수 있는 개수를 count 변수에 더하고, K에 동전을 사용하여 차감된 금액을 뺀다.
